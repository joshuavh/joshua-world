SYNG Karaoke

A new karaoke system for the Norwegian franchise.

Challenge


The karaoke bar franchise SYNG is all about having a fun night out. Unfortunately, their outdated karaoke system was not contributing to that. The goal for their new custom system was to be playful, unconventional but still so intuitive that even tipsy karaoke singers would have no problems using it.

Result


The new system covers all touch points necessary for a great karaoke experience. A back-end solution for admins to add new songs the music library, a booking system for SYNGâ€™s staff and lastly, a new and improved interface for the customers to find and play their favorite songs.
All touch points feature the bold and whacky look that makes SYNG so charming. The new touch screens enable customers not only to search for songs, but also explore categories and curated playlists. The system features audio-reactive animations and the increased button sizes also allow tipsy singers to find a song. Client: SYNG
